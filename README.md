# tarassum

A little web tool for drawing maps of the islamic world. You can pick cities, show routes between them, change styles, and export a png or pdf.


## running it

```
npm install
npm run dev
```

## build

```
npm run build
```


## stack

react, vite, leaflet, tailwind, react-router. place data comes from thurayya (https://github.com/althurayya/althurayya.github.io/). base layers are natural earth geojson.
